# ApeX
ApeX, a tool that uses social engineering to get WIFI keys.

### Usage
```sh
 python apex.py <interface>
```

### Requirements
- Kali Linux
- Python V2.7

### Installation
```sh
chmod +x install.sh apex.py
./install.sh
```

### Usage
```sh
./apex.py <iface>
```

### Demo
```sh
./apex.py wlan0
```
